# Claude Code Plans

このディレクトリには、Claude Codeによって生成されたプランファイルが保存されます。

## ディレクトリ構造

- `plans/` - 実装計画ファイルが保存されるディレクトリ
- プランファイルは自動的に生成され、タスクの実装計画を管理します

## 使用方法

Claude Codeがプランモードに入ると、このディレクトリ内にプランファイルが自動的に作成されます。
プランファイルには以下の情報が含まれます：

- タスクの概要
- 実装手順
- 影響を受けるファイル
- 考慮事項

## 注意事項

- プランファイルはバージョン管理に含めることができます
- 他の開発者と実装方針を共有する際に役立ちます
